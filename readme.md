# EcoPlein
Site de consultation des prix des carburants en France.

### Auteurs
SADLI Agnies
L'HONORÉ Eliott

### URL
https://lhonore.arbusmegacorp.top

https://sadli.arbusmegacorp.top

### Fonctionnalités
- Navigation par carte interactive des régions de France.
- Recherche de stations par ville et type de carburant (SP95, SP98, Gazole, E10, GPL, E85)
- Stations à proximité via géolocalisation
- Statistiques des villes les plus consultées
- Page technique : film Ghibli aléatoire (API JSON) + position géographique (API JSON)

### Ressources

PHP 8, HTML5, CSS3

API gouvernementale des prix des carburants : https://data.economie.gouv.fr/explore/dataset/prix-des-carburants-en-france-flux-instantane-v2/api/
API ipinfo.io — géolocalisation IP : https://ipinfo.io/developers/lite-api
API Ghibli — films Studio Ghibli : https://ghibliapi.vercel.app/

Données CSV INSEE (communes, départements, régions)
